Joe's PC Security Audit

A complete cybersecurity audit and hardening report for a Windows 10 PC, completed as part of Udacity's "Cybersecurity Foundations" Nanodegree (Level Three), delivered in partnership with the Digital Egypt Clubs Initiative (DECI).
Scenario

Acting as a hired security auditor, I was asked to assess and secure a "broken" Windows 10 PC used by a small business (Joe's Auto Body) that had been used insecurely and was suspected of being compromised.
What the Audit Covered
1. Reconnaissance
Documented the PC's hardware specs, installed software, and running applications.
Reviewed system and security service statuses (firewall, antivirus, security warnings).
Catalogued all user accounts and their access levels.
2. Assessment
Recommended an authentication strategy based on the three factors of authentication (something you know / have / are), proposing multi-factor authentication (MFA).
Explained the security value of enabling the Windows Firewall and Microsoft Defender Antivirus.
3. Securing Access
Proposed access control recommendations following the principle of least privilege for different user roles (owner vs. assistant).
Explained the risks of unused accounts and excessive administrator privileges.
4. Securing Applications
Identified unnecessary and non-compliant applications (games, unauthorized network scanning tools).
Identified outdated software posing security risks.
Extended the least-privilege model to two additional employee roles (bookkeeper, head mechanic), including tailored access and authentication recommendations.
Analyzed the risks of a suspected compromised employee email account and proposed an incident response.
Key Concepts Applied
Systems reconnaissance and asset inventory
Multi-factor authentication (MFA)
Principle of least privilege
Firewall and antivirus configuration
Risk identification and mitigation
Incident response basics
